About Sino.

Sino – file manager program like Total Commander.

There are differences of the program:

♦ Panels number – from 1 to 4.

♦ Direct link via TCP/IP.

♦ Fast search organization then other analogs.

♦ Creating folders in the archive directories.

♦ Menu utils plugins.

♦ Search plugins.

♦ Simple plugin install way (put to the <PLUGINS> folder).

♦ Using C++ and extern “C” library.
