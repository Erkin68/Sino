► Чем отличается программа от других аналогичных программ?

Некоторые отличительные черта программы “SINO”:

            - изменение количества панелей от 1-го до 4-х.   Конфигурация -> Изменить -> Общие параметры -> Панели;

            - возможность расположения автоматической замены рисунков в фоне панелей;  Конфигурация -> Изменить -> Цвета -> Для панелей -> i-панель -> Использовать фон;

- исключения установочных программ и других механизмов при работе плагинов, использовав при этом лозунг “бросил плагин в необходимою папку и работай”;

- проектировать работу плагинов “виртуальная панель” таким образом, чтобы  максимально виртуализировать процесс подключения к панелям;

- сравнительно наибольшая скорость в миллисекундах времени выполнения при использовании внутренного плагина поиска – <SinoSearch.dll>; Alt + F7;

- высокоскоростная сортировка по размеру;

- никогда не использовать реестр Windows для хранения конфогурационной или другой информации, для этого использовать диск(файлы);

            - включение  наибольшего количества использованных операции при работе с архиваторным ZIP-плагином . Например, создание папки, удаление файла или директории, и многое другое. Польное задейтствие всех операций автор не наблюдал ни в одном другом архиваторе файловых менеджеров;

            - максимально приблизить работу других архиваторных плагинов до уровня работы ZIP-плагина. (cab,exe,rar,chm,iso,nrg,…);

            Для работы  архиваторного плагина необходимо создать плагин в соответствии с требованиями (см. раздель справочной информации SinoPlgWrtrsHlp32.exe) и скопировать этот файл (*.dll) в папку “Plugins\Archive”. При этом можно скопировать целые папки. “SINO” сам найдёт небходимый плагин и задействует его в нужном месте. Механизмы работы других плагинов работают аналогичным образом. Такой подход задействия плагина автор наблюдал при работе с “Autodesk 3DS MAX” и “Adobe Photoshop”.

                Вопрос создания других плагинов остаётся открытым. По Интернету можно закачать открытый код необходимого плагина или его аналога, на основе этого кода можно создать свой плагин.

            Программа “SINO” и все его модулы автором был написан на языке “Microsoft Visual C++, 2008”, стандартные подпрограммы для стыковки плагинов написаны с использованием extern “C”. Этот факт значительно улучшает параметры программы и даёт широкие возможности интузиастам-создательям плагинов.

            Автор надеется на широкое распространение программы на международном масштабе и нахождения своего достойного места среди других аналогичных программ. Пусть программа “SINO” поможет поднять имедж  “Востока”  в современной программировании.
